# Structura

**Structura** is a lightweight Windows desktop application that converts a plain-text folder outline into a fully generated project structure on your computer — instantly.

Paste your structure, preview it live, then create all folders and empty files with a single click.

---

## What Structura Does

Structura helps you avoid repetitive manual setup by generating complete directory trees from simple text input.

It is designed to be:
- Fast
- Offline
- Simple
- Professional
- Desktop-friendly

No installers, no dependencies — just run the executable.

---

## Key Features

- Text-based project structure generation
- Real-time visual preview before creation
- File-type icons for better clarity
- Clean dark UI optimized for Windows
- Safe local file creation only
- Portable single EXE application

---

## How to Use

### 1. Run the application

### 2. Paste your structure
Use **2 spaces per indentation level**.

- Names **without extensions** → folders  
- Names **with extensions** → empty files  

### Example input:
```text
MyProject
  Assets
    Images
      logo.png
    Fonts
  Resources
    Themes
      Colors.xaml
      Typography.xaml
  UI
    Views
      MainView.xaml
      LoginView.xaml
    ViewModels
      MainViewModel.cs
